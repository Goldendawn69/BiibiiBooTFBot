A cute non-serious currency could exist.

Examples:

Sparkles
Buttons
Tea Biscuits
Moon Tokens
Glitter Dust

Earned by playing, completing bingo, participating in events.

Spent on harmless things:

Reroll form
Lock favourite form for a day
Trigger random event
Buy silly title
Summon a pair event request
Add temporary modifier

This can be fun, but be careful. Economy systems become spreadsheet beasts quickly. Tiny economy only. No grind. No pay-to-win. No real money.
