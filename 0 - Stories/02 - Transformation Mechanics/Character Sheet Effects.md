Once base characters exist, transformations can react to the character profile.

Example:

If base form says “witch”, magical transformations get extra flavour.
If a stat like Mischief is high, random events are sillier.
If Willpower is high, resistance rolls get a bonus.
If Glamour is high, glamour private flavour gets more confident.

This fits your base character idea nicely. The story file for base characters is already in Player State Upgrades.

Why it is good: it makes players feel seen without needing AI generation.
