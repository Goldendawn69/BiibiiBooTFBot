Instead of always changing the whole form, random events can add a temporary modifier to the current form.

Examples:

“Extra Fluffy”
“Pocket-Sized”
“Sparkle-Cursed”
“Fancy Hat”
“Dramatic Entrance”
“Tea-Scented”
“Overly Polite”
“Glamour Glow”

A player could be:

Living Plush Bunny + Extra Fluffy
Boutique Mannequin + Dramatic Entrance
Haunted Teapot + Overly Polite

Your Random Event idea already hints at this with “everyone currently transformed gets a fancy hat” and “all plushie forms become extra fluffy for the next hour”. This is worth making into a real system.

Why it is good: it creates funny ongoing server moments without needing a new image for every variant.
Implementation shape: store activeModifiers on users with an expiry time.
