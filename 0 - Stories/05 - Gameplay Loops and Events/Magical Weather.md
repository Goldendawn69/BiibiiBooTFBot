Add a server-wide state that changes how transformations are described for a short time.

Examples:

“Glitter Storm”
“Moonlit Mischief”
“Plush Hour”
“Mannequin Display Night”
“Teapot Tempest”
“Glamour Surge”
“Tiny Trouble”

During “Plush Hour”, plushie transformations are more common. During “Glamour Surge”, glamour-tagged forms get extra stylish private flavour. During “Tiny Trouble”, random events favour tiny forms.

Why it is good: the server feels alive.
Implementation shape: store a single serverState JSON file or per-guild state later.
